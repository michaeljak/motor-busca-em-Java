Google tipo de motor de busca em Java

Um aplicativo é desenvolvido em Java utilizando interfaces gráficas, ou seja, componentes Swing visuais.
O aplicativo permite que você registre funcionários e principais informações sobre eles. Em seguida, 
você pode procurar por funcionários em tempo real de acordo com seus sobrenomes paternos.
 Isto é, como o nome é lista de resultados que se infiltra jogo escrito.
 A lista de resultados é atualizado cada vez que uma tecla é liberada, usando o evento correspondente.
Este diz que não precisa clicar em um botão, mesmo que você precisa para pressionar enter.
O melhor é que não devemos ser a entrada de dados cada vez que você executar o programa, 
já que os dados são salvos em arquivos e carregado cada vez que o programa é aberto,
 o que torna possível unir uma boa lista, ao contrário do programas que não fazem uso de arquivos.
